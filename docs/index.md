## What is Lemma?

Lemma is an open-source, AI-native GRC platform that treats compliance as a rigorous, provable engineering discipline. By unifying compliance workflows into Git, adopting the NIST OSCAL standard, and introducing mathematically transparent AI mapping paths, it eliminates the black-box "compliance theater" common in traditional audit systems.

### Core Architecture
- **Parse:** Intelligent ingestion of PDFs, spreadsheets, and control catalogs via Docling.
- **Index:** Vector embeddings stored in ChromaDB for high-accuracy standard retrieval.
- **Map:** Traceable, score-weighted mapping of proprietary policies to baseline OSCAL frameworks via AI.
- **Harmonize:** The Common Control Framework (CCF) layer that normalizes shared controls across boundaries — *Test Once, Comply Many*.
