# Getting Started

> **Note:** The Lemma CLI and Core Engine are currently in active development (Phase 1).

## Installation

Once released, you will be able to install the Lemma CLI globally:

```bash
npm install -g @lemma-ai/cli
```

## Initializing a Repository

To scaffold the compliance tracking configuration in your existing local infrastructure project:

```bash
lemma init
```

This will generate a `.lemma` directory containing your local graph index, active framework lists, and connector manifests.
